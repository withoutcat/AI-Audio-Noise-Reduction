Adds or updates the video effect for the specified node ID and template.

Since
v4.6.0
Note
Priority rules:
  • The VIDEO_EFFECT_NODE_ID::STYLE_MAKEUP node takes precedence over the VIDEO_EFFECT_NODE_ID::FILTER parameter.
  • To apply the VIDEO_EFFECT_NODE_ID::FILTER parameter, you must first remove the VIDEO_EFFECT_NODE_ID::STYLE_MAKEUP node:
    removeVideoEffect(VIDEO_EFFECT_NODE_ID::STYLE_MAKEUP);
    addOrUpdateVideoEffect(VIDEO_EFFECT_NODE_ID::FILTER, "template name");
