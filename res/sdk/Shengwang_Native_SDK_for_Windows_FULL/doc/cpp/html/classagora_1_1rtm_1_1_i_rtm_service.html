Creates an RTM channel.

  • The space character.
  • Punctuation characters and other symbols, including: "!", "#", "$", "%", "&", "(", ")", "+", "-", ":", ";", "<", "=", ".", ">", "?", "@", "[", "]", "^", "_", " {", "}", "|", "~", ","
eventHandlerThe pointer to IChannelEventHandler.

  • An empty pointer NULL, if the method call fails.
